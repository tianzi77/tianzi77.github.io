jQuery TAB组件

  • 使用方法
  • event
  • 触发TAB的事件类型，鼠标悬停：mouseover，鼠标点击：click，默认为mouseover。
  • timeout
  • 事件延迟，单位为毫秒，默认为0。
  • auto
  • 自动切换，单位为毫秒，默认为0。
  • callback
  • 回调函数，触发TAB时执行，函数的参数返回的是this。
